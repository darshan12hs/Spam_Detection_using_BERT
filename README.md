# Spam Detection using BERT

The goal of this project is to understand how pre-trained transformer models like BERT can be used for text classification tasks such as spam detection.

The project includes implementations for:
- SMS spam classification  
- Email spam detection  
- Working with pre-trained BERT models  

### SMS Spam Detection
- Built a spam classifier for SMS messages using BERT  
- Applied text preprocessing and tokenization  
- Used transformer-based embeddings for classification  

### Email Spam Detection
- Developed a model to classify emails as spam or not  
- Explored NLP techniques and model performance  

### BERT Pre-trained Model
- Demonstrated how to use pre-trained BERT models  
- Used Hugging Face Transformers for inference  
- Tested classification tasks without extensive training  

## Key Concepts

- Transformer architecture  
- BERT model  
- Tokenization  
- Text classification  
- Pre-trained models
